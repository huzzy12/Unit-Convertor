# Premium Unit Converter

A beautiful and interactive unit converter application built with Python, Streamlit, and UV (Pint).

## Features

- Convert between various units across different measurement categories
- Interactive and visually appealing UI with animations
- Graphical representation of conversion results
- Responsive design for all devices

## Installation

1. Clone this repository
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the Streamlit application:
   ```
   streamlit run app.py
   ```
2. Open your browser and navigate to the URL displayed in the terminal (usually http://localhost:8501)
3. Select a conversion category, input a value, and see the results!

## Dependencies

- streamlit: Web application framework
- pint: Unit conversion library
- plotly: Interactive visualization
- streamlit-lottie: Animation integration
- requests: HTTP library for fetching animations
- streamlit-option-menu: Custom navigation menu "# Unit-Convertor"
